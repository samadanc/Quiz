Title: Bookmark questions

Primary Actor: User

Stakeholders and Interests:

User: wants to review specific questions without searching

Precondition: There is a list where the question are be added

Trigger: User selects question to be bookmarked

Main Scenario:

  1. User returns to a finished quiz
  2. User selects a question they had difficulty with
  3. SuD adds question to the list of bookmarked questions

Extensions:

2a. User selects a question they got incorrect

3a. Remove question from list

3a1. User goes to list to see all bookmarked question

3a2. User selects a question they wish to remove

3a3. SuD removes question from the list of bookmarked questions
