Title: Pause Quiz

Primary Actor: User

Stakeholders and Interests::

Instructor - Has the option to set a time to allow users take bathroom breaks.

User - Wants to be able to pause the quiz for X secs to take bathroom breaks.

Precondition: User already started the quiz

Trigger: User needs to use the bathroom or do something urgent

Main Scenario:

  1. user clicks on the Pause botton
  2. SuD verifies pause function availability
  3. SuD verifies time limit for quiz pause
  4. SuD confirms user request and pauses the quiz

Extensions:

1. Web Browser stop responding

2. SuD pause function not properly implemented with a time limit

3. Computer/server shuts down or crashes
