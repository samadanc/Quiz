Title: New Tab Report

Primary Actor: User

Stakeholders and Interests:

Instructor: wants to know the validity of quiz marks.

Technician: wants to recieve the number of times the user left the page.

Precondition: User is currently attempting a quiz.

Trigger: User opens a new tab within the browser.

Main Scenario:

  1. User is taking a quiz
  2. User leaves the quiz page
  3. SuD records the leaving of the page
  4. repeat 1-3 until quiz is finished
  5. lists the number the times the user left the quiz page

Extensions:

There are currently no extensions for this use case.
