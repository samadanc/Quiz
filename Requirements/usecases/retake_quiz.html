Title: Retake Quiz (No credit)
Primary Actor: Student
Stakeholders and Interests:

Student: Wants to use quiz for test practice
Instructor: Wants to allow students access to already taken quizzes

Precondition: Students have already completed the selected quiz they want to retake
Trigger: Quiz practice option available on selected quiz page

Main Scenario:

1. Student logs into quiz website
2. SuD shows student all available quizzes
3. Student chooses already taken quiz
4. SuD shows option to retake quiz (With no credit)
5. Student begins quiz
6. SuD lists available questions
7. Student chooses answers for quiz
8. Student submits quiz
9. SuD confirms quiz submission

Extensions:

1a. Student has not yet completed quiz for credit

1a1. Option to practice quiz will not be shown
1a2. Student instead goes on to do quiz for credit

2a. Student has not completed in-progress quiz

2a1. SuD displays message saying quiz has not been completed
2a2. Student continues off previous answers if saved
