Title: Accessibility

Primary Actor: User

Stakeholders and Interests:

Users: standard accessibility options to make system navigation easier without specialized hardware.

Trigger: Users Log into the system. The user updates Accessibility option (System Settings) based on their requirement

Main Scenario

  1. User toggles on Big Fonts
  2. SuD renders all the font 200% bigger
  3. User toggles on Color Intervsion
  4. SuD inverts all the colors
  5. User toggles on Voice Guidance
  6. Screen reader turns on to assist navigation and in quiz.

Extensions:

1a. SuD renders all the fonts on normal size

3a. SuD renders webpage in the original form

5a. SuD turns off this feature
