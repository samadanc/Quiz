Title: See Average

Primary Actor: Student

Stakeholders and Interests:

Student: Wants to view their average grade among all submitted quizes

Tech: Wants a record of student averages

Precondition: Student logged in to SuD

Main Scenario:

  1. Student navigates to screen of SuD where average can be viewed (Where should we put it?)
  2. SuD displays the student's average

Extensions:

2a. Student has not taken any quizes

2a1. The display for the average instead displays a message saying that no quizzes have been submitted

2b. The student's average is below the passing mark as defined by the instructor

2b1. A message is displayed along with the average stating what the required passing grade is
