Title: Highest score will be recorded as a final grade

Primary Actor: User and instructor

Stakeholders and Interests:

User: wants the highest attempt score to be recorded as the final grade

Instructor: wants to record and display the highest score in the gradebook

Precondition: A quiz allows multiple attempts

Trigger: A submission is done by user

Main Scenario:

  1. User submits the quiz for the first time
  2. SuD updates the first attempt score as the current highest score
  3. User submits the quiz for multiple times
  4. SuD computes and records the highest score among all the submission.
  5. SuD displays the highest score as the user's final grade

Extensions:

3a. No submission made.

3a1. SuD computes highest score from the previous submissions.
