Title: Logout

Primary Actor: User

Stakeholders and Interests:

User - Wants to logout with their session deleted

Technician - Wants activity log

Precondition: The user has logged in with correct credentials

Trigger: User clicks log out button

Main Scenario:

  1. User selects logout
  2. SuD displays yes/no prompt asking for confirmation of logout
  3. User selects yes, confirming logout
  4. SuD returns to login page, the session is then deleted

Extensions:

3a. User does not want to logout:

3a1. User selects no, refuting logout request

1a2. SuD resumes session normally
