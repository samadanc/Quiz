Title: Provide Solutions

Primary Actor: Student

Stakeholders and Interests:

Instructor - Wants to provide students with correct quiz answers

Student - Wants to receive correct answers upon quiz completion

Technician - Wants log of quiz submissions and quiz solutions

Precondition: The deadline for submission has passed

Trigger: Student wants to see correct solutions

Main Scenario:

  1. Student selects quiz in which they want to review correct answers
  2. SuD lists actions the student can take with reference to selected quiz
  3. Student selects option to review quiz
  4. SuD displays the quiz with each question showing the correct answer specified

Extensions:

1a. Student wants to view the answers of a specific quiz:

1a1. SuD displays list of completed quizzes in which student can pick from

1a2. Student advances to the specific quiz they want to view

4a. Student tries to modify answers:

4a1. Student cannot modify any elements of the quiz

4a2. Course instructor can modify quiz

4b. Student did not complete quiz:

4b1. Student gets all quiz answers wrong

4b2. Student can view correct quiz answers

4c. Student wants to compare their answer to the correct answer:

4c1. SuD displays students answer and mark for each question along with correct answer

4d. Student wants to view class average

4d1. SuD displays class average
