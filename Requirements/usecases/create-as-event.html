Title: Create a New Event

Primary Actor: Student

Stakeholders and Interests:

Student: wants to build the quiz as an event in calendar app

Tech: wants statistics of calendar app use

Precondition: Student logged in to quiz selection page.

Main Scenario:

  1. Student enters the page of a specific quiz.
  2. SuD presents create as event button on the page.
  3. Student clicks create as event button.
  4. SuD presents a list of apps to be used.(e.g. Google calendar, MS outlook)
  5. Student chooses his app.
  6. SuD navigates to the app with essential form data predefined.
  7. Student finished event creation in the app.

Extensions:

4a.Some new calendar app is supported by system

4a1.SuD presents those options in the app list

4a2.Student is able to use that one

4b.Some existed calendar app is unavailable now

4a1.SuD disables those options
