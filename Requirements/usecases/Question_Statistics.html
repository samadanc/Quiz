Title: Trends in Marking

Primary Actor: Instructor

Stakeholders and Interests:

Instructor: Can see trends in which questions students performed poorly on, and can review in class

Student: May recieve further guidance in class for similar questions

Precondition: User is logged in, and has permission to view submissions.

Trigger: Instructor requests to view class submission data.

Main Scenario:

  1. Instructor would like to see where students are making mistakes
  2. Instructor requests quiz submission statistics
  3. SuD determines which questions were answered incorrectly most frequently
  4. SuD returns a list of questions in decending order of percentage marked incorrectly

Extensions:

2a.Instructor choses the specific quiz they want the statistics of

4a.List is returned in decending order with the highest frequency incorrect questions appearing first
