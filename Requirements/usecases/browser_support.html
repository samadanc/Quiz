Title: Browser Support

Primary Actor: User

Stakeholders and Interests:

User - Wants page to render and act correctly across different devices, enviroments, and browsers

Trigger: User opens the site on their device

Main Scenario:

  1. User opens quiz site
  2. SuD responds with page to user
  3. User views a element on the returned page
  4. SuD renders as the developer of that element intended regardless of the modern browser being used
  5. User interacts with a element on the returned page
  6. SuD responds with a action the developer of that element intended regardless of the modern browser being used

Extensions:

4 & 6. Modern web browser can be defined as a browser that is:

  • Running a version that is supported by its vendor (i.e. Mozilla, Google)
  • Accounts for at least 0.5% of the global usage of browsers (such as following the data found on https://caniuse.com/usage-table)
