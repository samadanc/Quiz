Title: Device Adaptability

Primary Actor: User

Stakeholders and Interests:

User : wants to access the website through any size device.

Precondition: User has a device that can access to the internet.

Trigger: User tries to access to the website through different sizes screen devices.

Main Scenario:

  1. User accesses the website.
  2. SuD recognizes the size of the device that the user uses.
  3. SuD renders the website into a fit size over the user's device.
  4. User runs proper functions on the website.

Extensions:

1a. The website does not support the device.

1a1. The websites delivers an error message through a new windowbox.

Variation: 3a. The website adjusts its size upon the device's screen.
