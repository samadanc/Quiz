Title: Quiz Hint

Primary Actor: User

Stakeholders and Interests:

Instructor - has the option to either include or not, hints along with some/all quiz questions

User - wants to be able to take a loss on credit for the sake of a better chance at the correct answer

Trigger: User doesn't know/isn't sure of a question answer

Main Scenario:

  1. Instructor opts to offer a question hint
  2. Instructor defines the hint when creating a quiz question, and defines the amount of partial credit to be recevied for the correct answer after using the hint
  3. User reads question
  4. User either is unsure or does not know answer
  5. User opts to sacrifice some defined percentage of the question grade for a question hint
  6. SuD responds by providing a hint
  7. User answers the question
  8. SuD provides either partial or no credit depending on answer

Extensions:

2, 3. Different types of questions could neccesitate various forms of "hint":

  • Hint for fill in the blank type questions could be additional text/diagram
  • Hint for multiple choice type questions could be the removal of some amount of wrong selections
