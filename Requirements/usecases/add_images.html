Title: Add Images

Primary Actor: User with Editing Permissions (User)

Stakeholders and Interests:

User with Editing Permissions: wants to add images to a quiz question

Student: wants to recieve images associated with a quiz question

Technician: wants changes to be trackable

Precondition: User logged in already, create/edit quiz page open.

Trigger: User requests to add an image to a quiz question.

Main Scenario:

  1. User selects desired image(s) to upload to SuD.
  2. User local device uploads images to SuD, which stores and keeps track of them.
  3. Confirmation message is shown by SuD to verify images were added successfully.
  4. Image(s) is/are displayed to the user alongside the question to which they were added.
  5. User can repeat steps 1-3 until all images are added, then saves quiz/question.
  6. Any added images will be displayed to student while taking the quiz

Extensions:

5a. User can remove images from questions

5a1. User can interact with the SuD to remove an image from a qusestion

5a2. SuD removes image from question
