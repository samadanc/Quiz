Title: Flagging Quiz Questions

Primary Actor: Student

Stakeholders and Interests:

   Student - Wants flag notifications

   Instructor: Wants to enable flag notifications for students.

Precondition: Flagging must be enabled for quiz.

Trigger: Student wants to flag selected question and return to the question for whatever reason.

Main Scenario:

  1. Student comes to a question they are unsure of and selects flag option
  2. SuD displays symbol next to question selected.
  3. Student skips flagged question and proceeds.
  4. Student returns to flagged questions to try again.

Extensions:

1a. Student can toggle flag on and off.

a1. Student clicks currently displayed symbol

a2. SuD no longer displays flag.
