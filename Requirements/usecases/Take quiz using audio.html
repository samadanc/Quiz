Title: Take a quiz using voice command.

Primary Actor: Student

Stakeholders and Interests:

Student: wants the quiz recorded

Instructor: wants quiz submission

Tech: wants log of quiz submission

Precondition:

User with Student privileges logged in already.

Student doesn't want to use his keyboard or mouse for the quiz.

The student's computer audio system is in good shape.

Main Scenario:

  1. System presents quizzes available to Student
  2. Student selects the quiz
  3. Program presents two options, audio system or normal system(using keyboard and mouse)
  4. Student selects audio system
  5. The system prompts the student to verify if the microphone is in good shape by conducting a test.
  6. The system displays the quiz commands at the top corner
  7. Student starts quiz
  8. System presents quiz question
  9. Student enters answer using the voice commands
  10. System advances to next question
  11. Student repeats 7-10 as desired
  12. Student submits quiz solution using voice command
  13. System confirms quiz submission using voice command

Extensions:

6a. Suspend Quiz

6a1. Student suspends quiz session using voice command

6a2. System maintains answers for subsequent quiz session

10a. Submission rejected

10a1. SuD says quiz incomplete (voice)

10a2. Student forces incomplete submission using voice command

Variation: 6b. Student selects any other quiz question using voice command

Variation: 6c. Student changes previous answer using voice command
